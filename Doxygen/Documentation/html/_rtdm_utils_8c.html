Populates the stream header.

This function populates a stream header in preparation for either sending a stream over the network or writing to a file. Each member of the stream header is populated with the required information. sampleCount- the number of samples in the stream

dataSize- total amount of data bytes in the stream
currentTime- current time

BOOL VerifyFileIntegrity ( const char *  fileName)

Verifies a #.stream file integrity.

This function verifies that the final stream "STRM" section in the file has the correct number of bytes. IMPORTANT: There is no CRC check performed. It assumes that all bytes prior to the last stream are written correctly. This check is performed in case a file write was interrupted because of a power cycle or the like.

Parameters
fileName- path/filename of file to be verified
Returns
BOOL - TRUE if the file is valid or made valid; FALSE if the file doesn't exist
